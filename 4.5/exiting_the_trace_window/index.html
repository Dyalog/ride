    Exiting the Trace Window

    To close the Trace window:

    • click in the Trace window's tab
    • click in the Trace window's toolbar
    • use the Escape command ()
    • use the Quit command ()
    • press the Esc key

    When the Trace window is closed, the function within that Trace window is removed from the stack.
